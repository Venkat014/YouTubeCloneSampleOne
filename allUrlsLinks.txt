| Feature                              | Example Full API URL                                                                                                                                                                                                                                                     |
| ------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Search videos/channels/playlists** | `https://www.googleapis.com/youtube/v3/search?part=snippet&type=video&q=cricket&maxResults=10&key=API_KEY`                                                                                                                                                               |
| **Get video details**                | `https://www.googleapis.com/youtube/v3/videos?part=snippet,statistics&id=VIDEO_ID&key=API_KEY`                                                                                                                                                                           |
| **Get channel details**              | `https://www.googleapis.com/youtube/v3/channels?part=snippet,statistics&id=CHANNEL_ID&key=API_KEY`                                                                                                                                                                       |
| **Get playlists of a channel**       | `https://www.googleapis.com/youtube/v3/playlists?part=snippet,contentDetails&channelId=CHANNEL_ID&maxResults=10&key=API_KEY`                                                                                                                                             |
| **Get videos in playlist**           | `https://www.googleapis.com/youtube/v3/playlistItems?part=snippet,contentDetails&playlistId=PLAYLIST_ID&maxResults=10&key=API_KEY`                                                                                                                                       |
| **Get trending videos**              | `https://www.googleapis.com/youtube/v3/videos?part=snippet,statistics&chart=mostPopular&regionCode=IN&maxResults=20&key=API_KEY`                                                                                                                                         |
| **Get video comments**               | `https://www.googleapis.com/youtube/v3/commentThreads?part=snippet&videoId=VIDEO_ID&maxResults=20&key=API_KEY`                                                                                                                                                           |
| **Get comment replies**              | `https://www.googleapis.com/youtube/v3/comments?part=snippet&parentId=COMMENT_ID&key=API_KEY`                                                                                                                                                                            |
| **Get subscriptions**                | `https://www.googleapis.com/youtube/v3/subscriptions?part=snippet,contentDetails&mine=true&key=API_KEY` *(requires OAuth)*                                                                                                                                               |
| **Upload a video**                   | `https://www.googleapis.com/upload/youtube/v3/videos?part=snippet,status&key=API_KEY` *(requires OAuth)*                                                                                                                                                                 |
| **Analytics reports**                | `https://youtubeanalytics.googleapis.com/v2/reports?ids=channel==CHANNEL_ID&startDate=2024-01-01&endDate=2024-12-31&metrics=views,likes,subscribersGained&key=API_KEY` *(requires OAuth)*                                                                                |
| **Live streaming**                   | - Create broadcast: `https://www.googleapis.com/youtube/v3/liveBroadcasts?part=snippet,contentDetails,status&key=API_KEY` <br> - Get live chat: `https://www.googleapis.com/youtube/v3/liveChat/messages?liveChatId=LIVE_CHAT_ID&part=snippet,authorDetails&key=API_KEY` |
| **Embed & control video player**     | `<iframe width="560" height="315" src="https://www.youtube.com/embed/VIDEO_ID?enablejsapi=1" frameborder="0" allowfullscreen></iframe>`                                                                                                                                  |
